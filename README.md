## About me

I'm a Full Stack Developer with experience in backend development using **PHP (Laravel)**, **Node.js (NestJS)** and **Java (Spring Boot)**, as well as frontend applications with **React**, **Vue.js** and modern web technologies. I work with **RESTful APIs**, **microservices**, **queue-based processing**, **Redis**, **relational and non-relational databases**, always applying good practices such as **SOLID**, **Clean Code**, **GitFlow** and agile methodologies.

Currently, I work on the development and maintenance of scalable systems, performance optimization, database refactoring, test coverage improvement, and secure production-ready applications. In recent projects, I contributed to reducing API response times by up to **90%**, processing times by around **94%**, and increasing automated test coverage from **0% to 70%**.
